Command Guide

Use voice commands to manage services.

Basic Commands

  • Restart a service
    "Restart [service_name]"
  • Scale a service
    "Scale [service] to [N] replicas"
  • Get Status
    "Status of [service]"
  • Get Logs
    "Show logs for [service]"

Examples

  • "Restart payment gateway"
  • "Scale checkout to 5"
  • "Status of database"

Troubleshooting

If a command fails, try to be more specific with the service name.
